# Instagram DM ダッシュボード - 最終状況レポート

## プロジェクト概要

Instagram DMダッシュボードは、InstagramのダイレクトメッセージをAI生成のインサイトと統合するNext.js 15.2.2アプリケーションです。このアプリケーションは、InstagramのDMを取得し、それに対するAI応答を生成して表示するダッシュボードを提供します。

## API統合状況

### Instagram API

1. **トークンタイプ**: ページアクセストークン
2. **プロフィール情報**: ✅ アクセス可能
3. **DM (会話)**: ✅ アクセス可能
4. **ビジネスアカウント**: ❌ 未連携（オプション）

### OpenAI API

1. **API接続**: ✅ 設定済み
2. **モデル**: GPT-3.5 Turbo（デフォルト）

## 実装機能

1. **Instagram DM取得**: InstagramのDMをリアルタイムで取得
2. **AI応答生成**: OpenAI APIを使用して応答を生成
3. **顧客管理**: 顧客ごとの会話履歴を表示
4. **会話表示**: 会話の詳細とAI生成の応答候補を表示
5. **設定管理**: API設定、表示設定、通知設定の管理
6. **APIステータス確認**: Instagram APIの接続状態と権限の確認

## トークンタイプ解説

### 1. ユーザーアクセストークン
- **説明**: 個人のInstagramアカウントに関連付けられたトークン
- **権限**: 基本的なプロフィール情報の取得が可能
- **制限**: DMへのアクセス権限（`read_mailbox`）は通常含まれない

### 2. ページアクセストークン
- **説明**: Facebookページに関連付けられたトークン
- **権限**: ページに関連するInstagramビジネスアカウントのデータにアクセス可能
- **制限**: ビジネスアカウントに関連付けられている必要がある
- **DMアクセス**: このトークンタイプがDMアクセスに最適

### 3. アプリトークン
- **説明**: アプリケーション自体に関連付けられたトークン
- **権限**: アプリケーションレベルの操作に使用
- **制限**: ユーザーデータへのアクセスには別途ユーザートークンが必要

## 現在の設定

現在のアプリケーションは、ページアクセストークンを使用してInstagram APIと連携しています。このトークンは、DMへのアクセス権限（`read_mailbox`）を持っており、会話データを取得できます。

## 推奨事項

1. **ビジネスアカウントの連携**: より多くの機能を利用するために、Instagramビジネスアカウントを連携することを推奨します。
2. **トークン管理**: ページアクセストークンは定期的に更新する必要があります（約60日ごと）。
3. **エラー処理**: APIエラー時に自動的にモックデータを使用するフォールバックメカニズムを実装しています。

## デプロイ情報

アプリケーションは現在、Vercelにデプロイされています。環境変数は`.env.local`ファイルで管理されており、デプロイ時にVercelの環境変数設定に追加する必要があります。
