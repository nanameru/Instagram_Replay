# Instagram API の制限事項と対応策

## 概要

Instagram DMダッシュボードアプリケーションの開発において、Instagram APIを使用して過去のDMメッセージを取得する際に直面する制限と、それに対する対応策をまとめました。

## 主な制限事項

### 1. アクセストークンの期限切れ

Instagram Graph APIのアクセストークンには有効期限があります。現在のトークンは期限切れになっています：

```
Error validating access token: Session has expired on Saturday, 15-Mar-25 10:00:00 PDT.
```

### 2. ビジネスアカウントの要件

過去のDMメッセージを取得するには、以下の条件を満たす必要があります：

- **Instagram Business Account**: 個人アカウントではなく、ビジネスアカウントである必要があります
- **Facebook Page連携**: InstagramビジネスアカウントはFacebookページと連携している必要があります
- **Facebook Developer App**: Facebook開発者アプリを作成し、必要な権限を設定する必要があります

### 3. 必要な権限

Instagram APIで過去のDMを取得するには、以下の権限が必要です：

- `instagram_basic`: 基本的なプロフィール情報用
- `instagram_manage_messages`: メッセージ管理用（DMの取得と送信）
- `pages_messaging`: Facebookページを通じたメッセージング用
- `pages_read_engagement`: ページ情報読み取り用

特に注意が必要なのは、`read_mailbox`権限は廃止され、現在は`instagram_manage_messages`を使用する必要があります。

### 4. アプリレビュー

一部の権限（特に`instagram_manage_messages`）を使用するには、Facebookのアプリレビューが必要です。これには以下が含まれます：

- アプリの詳細な説明
- 権限の使用目的の説明
- スクリーンキャストやデモビデオ
- プライバシーポリシー

## 対応策

### 1. モックデータの使用

API制限に対応するため、本アプリケーションでは以下の対応を実装しています：

- **リアルなモックデータ**: 日本語の会話データを含む現実的なモックデータを生成
- **フォールバックメカニズム**: API接続が失敗した場合に自動的にモックデータを使用
- **エラー処理**: 詳細なエラーメッセージとステータス情報を提供

### 2. API状態の監視

アプリケーションには以下の機能が含まれています：

- **トークン検証**: `/api/instagram/status`エンドポイントでトークンの有効性を確認
- **権限チェック**: 必要な権限が揃っているかを確認
- **ビジネスアカウント検証**: Instagram Business Accountが正しく設定されているかを確認

### 3. ビジネスアカウント設定ガイド

ユーザーがInstagram Business Accountを正しく設定できるよう、詳細なガイドを提供しています：

- **アカウント変換**: 個人アカウントからビジネスアカウントへの変換手順
- **Facebook連携**: FacebookページとInstagramビジネスアカウントの連携方法
- **権限設定**: 必要な権限の設定方法

## 今後の対応

Instagram APIを完全に統合するには、以下の手順が必要です：

1. **新しいアクセストークンの取得**: Facebook開発者ポータルから新しいトークンを取得
2. **ビジネスアカウントの設定**: InstagramアカウントをビジネスアカウントにしてFacebookページと連携
3. **アプリレビューの申請**: 必要な権限（特に`instagram_manage_messages`）の使用許可を申請
4. **長期トークンの取得**: 60日間有効な長期トークンを取得し、定期的に更新する仕組みを実装

## 結論

現在のアプリケーションは、Instagram APIの制限に対応するためのモックデータと堅牢なフォールバックメカニズムを実装しています。実際のAPIデータを使用するには、上記の手順に従ってInstagram Business Accountを設定し、必要な権限を取得する必要があります。
